## 二、核心诉求

**不要做一个"信息展示型网站"，要做一个让人走进去的「数字档案馆」。**

这不是展示实践成果的宣传页，而是一次**线上展览**。观众打开网站时，ta 不是在浏览信息，而是在走进一间存放着北科大与天大历史渊源的档案馆。

### 设计原则
1. **从概念出发，不是从模板出发** — 先定体验再定 UI
2. **每个栏目有自己的性格** — 档案库 ≠ 口述史 ≠ 影像志，交互形式应反映内容特质
3. **内容是最强的装饰** — 设计要为未来的老照片、视频、扫描件留出空间，而不是用 UI 装饰填满
4. **手机和电脑是不同的入口** — 不是简单的响应式缩放

## 八、设计方向参考（不必遵循，仅为激发灵感）

### 概念关键词
守火人、根脉、薪火相传、抢救历史、寻根溯源、钢铁摇篮、时间赛跑、矿冶血脉

### 队徽核心意象
翻开的书本、水波纹/河流（"脉"）、银杏叶、圆形印章感、蓝金配色

### 视觉意象建议
- 水纹/脉纹（"脉"的视觉隐喻，可贯穿全站做装饰元素）
- 旧纸张/档案质感
- 暖色射灯（档案室/博物馆灯光）
- 钢铁/金属冷峻感（北科大"钢铁摇篮"）+ 暖金点缀（历史温度）
- "数字存档"的现代感 — 不是仿古做旧，而是"用数字手段保存历史"

### 情绪调性
- 庄严但不沉重
- 温暖但有力量
- 历史感但有现代数字感

### 现有版本的问题（避免重蹈覆辙）
1. 布局太像管理后台（左侧栏 + 内容区 + 卡片网格）
2. 所有栏目都用同样的卡片组件，没有体现内容差异
3. 首页没有"入口感"，只是一个带标题的页面
4. 缺少情感钩子，打开网站没有"被吸引"的感觉
